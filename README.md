# LGS Portalı

LGS (Liselere Geçiş Sınavı) öğrencileri için kapsamlı başarı takip sistemi.

## Özellikler

- **Öğrenci Paneli**: Kişisel başarı takibi ve deneme sonuçları analizi
- **Öğretmen Paneli**: Sınıf performans analizi ve veri yönetimi
- **Analitik & Raporlar**: Detaylı performans analizleri ve raporlama

## Teknoloji

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Firebase (Firestore)

## Deploy

Bu site GitHub Pages üzerinde host edilmektedir.
